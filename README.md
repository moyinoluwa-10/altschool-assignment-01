# AltSchool Assignment 01

A simple personal portfolio website built with HTML and CSS, featuring multiple pages and interactive components.

## Features

- **About Me Page**: Personal introduction and goals
- **Calculator**: A functional calculator application
- **Login/Register Pages**: User authentication forms
- **Contact Page**: Contact form and information

## Project Structure

```
altschool-assignment-01/
├── index.html              # Main landing page
├── HTML Files/             # Additional HTML pages
│   ├── calculator.html
│   ├── contact.html
│   ├── login.html
│   └── register.html
├── CSS Files/              # Stylesheets for each page
│   ├── index.css
│   ├── calculator.css
│   ├── contact.css
│   ├── login.css
│   └── register.css
└── images/                 # Image assets
```

## How to Run

1. Clone this repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the navigation menu

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts
